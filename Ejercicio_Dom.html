innerHTML

Descripción: Permite obtener o establecer el contenido HTML de un elemento. Incluye etiquetas HTML, no solo texto.

textContent

Descripción: Obtiene o establece el contenido de texto de un elemento, ignorando cualquier HTML. No se interpretan etiquetas, solo se muestra el texto.

innerText

Descripción: Similar a textContent, pero tiene en cuenta el estilo CSS y el formato. Solo devuelve el texto que es visible para el usuario.

outerHTMl

Descripción: Permite obtener o establecer el contenido HTML del elemento seleccionado, incluyendo el propio elemento. Es similar a innerHTML, pero incluye la etiqueta del elemento en sí.
